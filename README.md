# SwiftData + CloudKit Sync Tutorial

Sample project for the Medium article:

“How to Enable iCloud Sync in Your App”

This project demonstrates:

• SwiftData models
• Automatic CloudKit synchronization

## Requirements

- Xcode 15+
- iOS 17+
- iCloud enabled device

## Setup

1. Clone the repo
2. Open the project in Xcode
3. Enable your Apple Developer Team
4. Run the app on a physical device
